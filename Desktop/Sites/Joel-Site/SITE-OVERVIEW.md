## 🎨 Design System

### **Colors**
```
Background:     #09090b (deep black)
Surface:        #141414 (dark gray)
Surface2:       #1a1a1a (lighter gray)
Text Primary:   #f2ece0 (cream)
Text Muted:     #a89f8c (tan)
Gold:           #c8a96e (primary accent)
Gold Light:     #e5d0a4 (hover state)
Border:         rgba(200, 169, 110, 0.2)
```

### **Typography**
- **Display**: Cormorant Garamond (headings, elegant)
- **Body**: Jost (clean, modern)
- **Template fonts**: Various (Playfair Display, Inter, Raleway, etc.)

### **Layout**
- Max width: 1280px (7xl)
- Padding: 5-6% horizontal
- Section spacing: 6rem (24) vertical
- Grid: Responsive (1 → 2 → 3 columns)
